## What I've Shipped

### NetSwipe — iOS Networking App `2025`

> *A swipe-based professional networking platform. Built solo from idea to working product.*

The premise: LinkedIn is a feed. Bumble BFF is personal. Neither is built for the moment when you're at a hackathon, a conference, or a campus event and want to find someone worth talking to. NetSwipe fills that gap.

**Engineering:** SwiftUI (iOS), Node.js/Express/MongoDB backend, Firebase OTP auth, real-time chat with avatars and timestamps. Designed in Figma from first wireframe to final prototype. Gamification layer built into core UX for engagement.

**Results:** 40% reduction in account setup time · 25% drop in login failures · 35% increase in active engagement

`SwiftUI` `Node.js` `Express` `MongoDB` `Firebase` `Figma` `Real-time Chat`

### AdMind — AI Ad Creative Intelligence Platform `Cal Hacks 12.0 · 2025`

> *Computer vision + LLMs applied to mobile ad creative analysis.*

Built an end-to-end pipeline that ingests ad creatives, extracts visual and emotional signals (objects, text, color, sentiment), and generates plain-language breakdowns of messaging quality, hook strength, and UX risk — using Gemini for the LLM layer.

**Result:** 60% reduction in manual ad-review time. Structured creative intelligence, not just raw classification.

`Python` `Gemini` `LLMs` `Computer Vision` `Analytics Pipeline`
